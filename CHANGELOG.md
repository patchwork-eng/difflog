# Changelog

All notable changes to this project will be documented in this file.

## v1.0.1 — 2026-03-15

### ✨ Features
- A new landing page for difflog.io has been added, which is now accessible via GitHub Pages. This enhances the visibility and accessibility of the project.
- The onboarding flow has been improved with the addition of a "Bring Your Own Key" (BYOK) trust section, providing users with clearer guidance on security practices.

### 🐛 Bug Fixes
- Error handling has been hardened to improve the robustness of the application. This includes a skeleton for the license and enhancements to the overall test coverage.

### 🔧 Chores & Maintenance
- The README file has been restored along with the `docs/CNAME`, which were accidentally dropped in an earlier push, ensuring that documentation is complete and accessible.
- A Jest test suite has been added to the project, which will help in maintaining code quality and addressing edge cases effectively.
